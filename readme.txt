========================
 パノラマストリーミング
 2015/12/08
========================
概要
Thetaカメラの映像をEquirectangularに変換して動画保存するアプリです。
ffmpegの出力を -f flv 配信先URL と指定することでパノラマのライブ配信ができます。

事前準備
[CameraDevices.txt]
接続を許可するカメラデバイスを名前で指定します。
(上から優先されます)

起動方法
ThetaLiveVR.exeから起動します。

キー操作
Rキー 再接続
Qキー 切断
Escキー 終了

動画保存先
ffmpegフォルダ内に保存されます。

注意事項
動画書き出し中にコマンドプロンプトが立ち上がります。
カメラデバイスの名前が分からない場合はカメラを接続後、アプリを一度起動して閉じます。
「ThetaLiveVR_Data」フォルダにある「output_log.txt」で確認できます。
また、ffmpegで入力する解像度はSetting.jsonに記述したサイズに合わせてください。
